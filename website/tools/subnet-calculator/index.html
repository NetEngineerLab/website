METHODOLOGY

Methodology

CIDR calculation

The prefix length creates a contiguous subnet mask. Network and broadcast addresses are derived with bitwise AND and wildcard operations.

/31 and /32

A /31 is treated as a two-address point-to-point subnet, while a /32 represents one host route.

VLSM allocation

Requirements are sorted from largest to smallest and aligned to the next valid network boundary before allocation.

IPv6 prefixes

IPv6 results use 128-bit integer arithmetic and show the network, last address, host bits and subnet count from the parent prefix.

Primary references: RFC 1918 · RFC 3021 · RFC 4632 · RFC 4291 · RFC 4193

FAQ

Frequently asked questions

How many usable hosts are in a /24?

A /24 has 256 total addresses and normally 254 usable host addresses.

Why does /31 show two usable addresses?

RFC 3021 permits both addresses on a point-to-point link, eliminating network and broadcast reservations.

What is a wildcard mask?

It is the bitwise inverse of the subnet mask and is commonly used in ACLs and routing configuration.

Does VLSM reserve gateway addresses automatically?

Each row has an editable reserve field. Use it for gateways, redundancy addresses, infrastructure or growth.

Can IPv6 addresses be counted exactly?

Yes. The calculator uses BigInt arithmetic and displays the full decimal address count.

Should a large broadcast domain be used just because it fits?

No. Consider failure domains, broadcast traffic, security zones, routing summarization and operational boundaries.
