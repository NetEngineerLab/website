# IP Subnet Calculator V1.0 Professional

## Functions
- IPv4 address + CIDR or subnet mask
- network, broadcast, first/last host
- total and usable addresses
- subnet mask and wildcard mask
- previous and next subnet
- address scope detection
- reverse DNS zone for octet-aligned prefixes
- binary representation
- IPv6 prefix, last address, exact address count and parent-prefix subnet count
- VLSM planner with per-subnet reserve
- save, copy, CSV and print/PDF
- Chinese and English pages
- mobile layout, SEO, PWA, GA4 and AdSense placeholders

## VLSM rule
Subnets are sorted from largest to smallest and allocated on valid network boundaries.

## Install
`website/tools/subnet-calculator/`
