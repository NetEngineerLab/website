# Install into NetEngineerLab

Copy this complete folder to:

D:\NETENGINEERLAB\website\tools\otdr-event\

Then update:

- `website/data/tools-catalog.js`
- `website/sitemap.xml`

Use `integration/tool-catalog-entry.json` and `integration/sitemap-entry.xml` as the prepared entries.

The English tool URL is:

`/tools/otdr-event/en/`
