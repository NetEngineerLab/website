# ONU RX Power Diagnosis V1.0.2 Professional

## 功能
- 快速实测光功率诊断
- 链路模型诊断
- 正常、弱光预警、严重弱光、过载、疑似无光
- 接收灵敏度余量
- 过载安全余量
- 理论接收光功率
- 实测与理论偏差
- 推算额外异常损耗
- 光纤、熔接、连接器和分光器损耗模型
- 故障原因和排查建议
- 中文/英文独立页面
- 保存记录、复制、CSV、打印/PDF
- SEO、PWA、GA4、AdSense预留

## 安装目录
website/tools/onu-rx-power/
