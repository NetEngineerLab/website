METHODOLOGY

计算方法与工程说明

物理链路损耗

光纤长度 × 衰减系数 + 熔接损耗 + 连接器损耗 + 分光器损耗 + 其他无源损耗。

总设计损耗

物理链路损耗 + 工程余量。

接收光功率

发送光功率 − 物理链路损耗。

健康状态规则

剩余预算 ≥ 3 dB：健康；0–3 dB：风险；小于 0 dB：超限。

重要说明

默认衰减和分光器损耗属于常用工程参考值。正式设计应以设备规格书、企业标准和现场测试数据为准。

FAQ

常见问题

总设计损耗和物理链路损耗有什么区别？

物理链路损耗只包含实际器件与线路损耗；总设计损耗还加入工程余量，用于覆盖老化、温度变化、维护连接和测量误差。

1310 nm、1490 nm、1550 nm该如何选择？

应按实际发送方向和业务波长选择。常见PON上行使用1310 nm，下行可能使用1490 nm或1550 nm，具体以系统规格为准。

分光器损耗为什么可以修改？

页面使用常见标称值。不同厂商、封装方式和工作波长会产生差异，正式设计应以器件规格书或现场测试值为准。

剩余预算多少比较安全？

一般建议至少保留3 dB工程余量。余量越小，链路对污染、弯曲、老化及温度变化越敏感。

为什么接收光功率正常但预算显示风险？

预算与接收门限是两个不同判断维度。当前发光功率较高时，接收功率可能正常，但设计预算仍可能缺少长期工程余量。

计算结果可以用于正式验收吗？

本工具适合规划、复核和故障分析。正式验收应结合光功率计、OTDR、设备规格和企业验收标准。
