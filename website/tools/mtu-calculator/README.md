# MTU & MSS Calculator V1.0 Professional

## Functions
- Base underlay MTU and target inner MTU
- IPv4 and IPv6
- TCP MSS and actual TCP data with options
- UDP and ICMP payload
- PPPoE, MPLS, IP-in-IP, GRE, VXLAN, IPsec ESP and WireGuard presets
- Editable encapsulation stack
- VLAN expansion and strict frame-cap modes
- Required underlay MTU and wire-frame size
- Windows and Linux ping/PMTU commands
- Fragmentation risk and IPv6 1280-byte warning
- Save, copy, CSV and print/PDF
- Chinese and English pages
- SEO, PWA, GA4 and AdSense placeholders
- Mobile layout

## Engineering scope
IPsec presets are editable estimates because padding and cipher-suite overhead vary. VLAN tags normally increase the wire frame rather than reducing configured IP MTU when the hardware supports tagged-frame expansion.
