# Bandwidth & Transfer Time Calculator V1.0 Professional

## Modes
1. Transfer time
2. Required bandwidth
3. Concurrent-user capacity
4. Sustained-rate data volume

## Functions
- decimal SI network-rate units
- decimal and IEC binary file-size units
- payload efficiency and allowed utilization
- concurrent transfers
- RTT and setup round trips
- required physical bandwidth
- per-user burst factor and maximum user count
- daily and 30-day data volume
- save, copy, CSV and print/PDF
- Chinese and English pages
- mobile layout, SEO, PWA, GA4 and AdSense placeholders

## Engineering note
The calculator is a planning aid. Real throughput can also depend on congestion control, packet loss, RTT, queueing, QoS, storage performance and application behavior.

`website/tools/bandwidth-calculator/`
