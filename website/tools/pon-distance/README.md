# PON Maximum Distance Calculator V1.0 Professional

## 主要功能
- 光功率预算最大距离
- 系统/协议距离上限
- 有效最大传输距离
- 规划链路距离验证
- 一级、二级分光器，默认均为1:8
- 光纤、熔接、连接器和其他无源损耗
- 工程余量
- 预计接收光功率
- 物理预算余量与设计预算余量
- 光预算限制 / 系统距离限制判断
- 健康、仅物理可达、预算不足、系统距离受限
- 中文/英文独立页面
- 保存记录、复制、CSV、打印/PDF
- SEO、PWA、GA4、AdSense预留
