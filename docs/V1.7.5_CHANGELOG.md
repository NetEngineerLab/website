# NetEngineerLab V1.7.5 更新说明

## 版本目标

V1.7.5新增生产性能与Lighthouse自动巡检，不增加工具，不修改现有12个工具的计算公式、输入输出或用户数据处理方式。

## 新增功能

1. 新增`Production Performance Monitor` GitHub Actions工作流。
2. 在V1.7.4生产在线巡检成功后自动执行，避免审计未完成部署的版本。
3. 每天北京时间08:45自动建立正式站性能记录。
4. 对4个代表页面各运行3次Lighthouse并采用中位代表结果。
5. 检查Performance、Accessibility、Best Practices与SEO分类分数。
6. 检查FCP、LCP、CLS、TBT与Speed Index。
7. 检查总传输量、JavaScript、CSS、图片体积和浏览器控制台错误。
8. 新增结构化`PRODUCTION_PERFORMANCE_REPORT.json`报告及GitHub步骤摘要。
9. 保存Lighthouse原始JSON和HTML报告30天，便于定位回退原因。
10. 新增不依赖Chrome的策略单元测试，覆盖正常、性能回退和缺失审计场景。

## 初始策略

V1.7.5先建立稳定生产基线并阻止明显回退。阈值保存在`tests/lighthouse/production.lighthouserc.json`，后续只根据多次稳定结果逐步收紧。
