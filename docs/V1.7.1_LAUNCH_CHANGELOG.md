# NetEngineerLab V1.7.1 launch-ready changelog

## Navigation

- Added working category links from the home page.
- Tool-directory filters now support `?category=` deep links.
- Added category counts, active state, URL persistence and result status.

## URL and SEO

- Migrated About, Contact, Privacy and Terms to clean directory URLs.
- Added those pages to the bilingual Sitemap.
- Added legacy `.html` redirects.
- Set 404 pages to `noindex,follow` and removed 404 canonicals.
- Added Organization, WebSite, ItemList and BreadcrumbList JSON-LD.
- Added consistent Open Graph and Twitter metadata.

## Deployment

- Added Cloudflare Pages `_headers` and `_redirects`.
- Added Vercel fallback configuration.
- Added preview-host `noindex` headers.
- Added security and cache headers.

## Operations

- Centralized GA4 and AdSense configuration; both remain disabled by default.
- Added `npm run prepare:launch`, `npm run audit:launch` and `npm run preview`.
- Added a production-like local preview server.
- Updated all multilingual and service-worker versions to V1.7.1.
