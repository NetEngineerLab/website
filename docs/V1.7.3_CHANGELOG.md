# NetEngineerLab V1.7.3 更新说明

## 版本定位

V1.7.3是正式上线验收版，不增加第13个工具，不修改12个工具的核心计算公式。

## 新增生产验收

1. 新增`production-acceptance.js`。
2. 自动检查36个Sitemap正式页面。
3. 自动检查8条旧`.html`地址301跳转。
4. 自动检查中英文404返回真实HTTP 404。
5. 自动检查安全响应头、缓存和Service Worker重新验证。
6. 自动运行7个计算引擎测试。
7. 自动检查全站本地链接、锚点、重复HTML ID和空链接。
8. 自动检查正式域名、Sitemap、robots.txt、GA4和AdSense配置。

## 新增远程验收

正式部署后可运行：

```text
npm run accept:remote -- --base=https://netengineerlab.com
```

用于检查线上36个页面、301跳转、404、安全响应头、Sitemap及`security.txt`。

## 部署稳定性

- Node.js锁定为22.16.0。
- 新增GitHub Actions生产质量门禁。
- 新增`.gitignore`，防止`website_backup`、ZIP和编辑器目录上传。
- 新增`.gitattributes`统一文本换行。
- 新增发布文件SHA-256清单。

## Cloudflare Pages

- 修正工具图片缓存匹配规则。
- 为`data`目录设置短缓存。
- 为所有工具Service Worker设置`max-age=0, must-revalidate`。
- 保留`pages.dev`生产及分支预览`noindex`规则。
- 增加`security.txt`短缓存规则。

## 未修改

- 12个工具数量不变。
- 中英文URL不变。
- 36个Sitemap正式URL不变。
- Wi-Fi页面V1.7.2显示修正保留。
- GA4和AdSense仍默认关闭。
