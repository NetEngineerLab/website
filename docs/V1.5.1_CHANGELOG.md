# V1.5.1 change log

## -48V battery calculator

1. Replaced equipment unit power with direct unit current.
2. Removed the direct-power load mode.
3. Load modes are now:
   - equipment current list
   - direct total DC current
4. Equipment totals are displayed in amperes.
5. Power is derived internally from the nominal battery-bus voltage.
6. Added migration for old locally saved rows containing `watts`.
7. Results now display design current and calculated design power together.

## Layout

1. Increased the calculator workspace width.
2. Gave more width to the input panel.
3. Removed the table minimum width that caused horizontal compression.
4. Added fixed, balanced column widths.
5. Switches to a single-column calculator below 1120 px.
6. Equipment rows become mobile cards below 650 px.
